API Key
🔒 Your key is stored only on this device in local storage. It is never sent anywhere except directly to Anthropic's API.
How to get an API key
1. Go to console.anthropic.com
2. Sign up or log in (free account works)
3. Click API Keys in the left sidebar
4. Click Create Key — copy it immediately
5. Paste it in the field above and tap Save Key
⚠️ New accounts get free credits to start. After that, usage is pay-as-you-go — scanning a receipt or recipe costs a fraction of a cent.
